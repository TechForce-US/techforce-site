## Product: SIS
The SIS was built initially for LFBI but has resell potential with other schools who use Moodle as an LMS.
The SIS uses Moodle as a datasource with additional features built on top of it. Features such as transcript generation and reporting. 
Other private schools that rely on Moodle as an LMS is the ICP. 

## Product: Sound Security 
Sound Security is a Zigbee based smart home security hub/panel that is compatible with thousands of off the shelf Zigbee devices and RTSP cameras. The system can be setup entirely by the user. This is a hardware product.
